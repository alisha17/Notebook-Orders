This application uses Python 2.7.12 version.

You can download it from official Python Website:
https://www.python.org/downloads/

Please access the application using command line.

After installing Python in your system, run the application using:

python main.py

And follow the steps as displayed on the command line.

* Assumed already available subject and box sizes are specified in 
  a csv file - records.csv
  Please update file according to different subjects and sizes 
  available to test the application.
